# Overmax v0.2.1 릴리즈 노트

> v0.2.0 이후 변경 사항

---

## 🚀 주요 신규 기능 및 변경 사항

### 🖥️ V-Archive 동기화 미업로드 경고 및 헤더 직접 업로드 추가
- 현재 플레이 중인 곡의 기록이 V-Archive에 동기화되지 않은 상태일 경우, 오버레이 헤더 영역에 노란색 경고 인디케이터가 표시됩니다.
- 해당 인디케이터를 클릭하여 동기화 창을 거치지 않고 오버레이 헤더에서 V-Archive로 기록을 직접 편리하게 업로드할 수 있는 버튼이 추가되었습니다.

### 🔍 메타 정보 인식 정확도 및 안정성 강화
- 곡명의 길이가 길거나 난이도 정보 매칭 형식이 달라 인식이 비정상적이던 특정 케이스들을 수정하고, 플레이 정보(곡명, 패턴 등)의 메타데이터를 정합성에 맞게 제대로 표시하도록 개선했습니다.
- DPI 배율이 적용된 환경에서 오버레이 위에 마우스가 올라왔는지 판정하는 감도 영역을 개선하여 마우스 패스스루 동작이 정상적으로 수행됩니다.
